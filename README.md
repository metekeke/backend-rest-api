# Backend REST API

## Features
- RESTful API architecture
- JWT based authentication
- PostgreSQL database
- Environment variables management
- Deployed on AWS EC2

## Tech Stack
- Node.js
- PostgreSQL
- JWT
- AWS EC2

## Setup
```bash
git clone https://github.com/metekeke/backend-rest-api.git
cd backend-rest-api
npm install
npm run dev
